POURQUOI LE FOSS ?

LIBERTÉ

Vous possédez votre code. Pas de surveillance cachée. Pas de backdoors. Pas de tracking.

TRANSPARENCE

Le code source est public. Chacun peut vérifier ce qui se passe vraiment.

COMMUNAUTÉ

Des développeurs du monde entier améliorent continuellement les outils.

COÛT ZÉRO

Gratuit à utiliser, modifier, et distribuer. Aucune dépendance envers une corporation.
